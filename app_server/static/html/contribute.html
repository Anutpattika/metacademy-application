Contributing to Metacademy

Metacademy is young, and our online system for content submission is not quite ready. Right now, the best way to contribute is to send feedback to feedback@metacademy.org, e.g.

  • Do you find Metacademy's interface intuitive?
  • Have you learned any concepts using Metacademy? If so, how was the experience?
  • Is there a specific feature you would like Metacademy to implement?
  • Did you find a bug or notice strange behavior when using Metacademy?

We want to make Metacademy as useful as possible, so we promise to (1) sincerely consider your feedback and (2) send you a follow-up response. -Colorado Reed & Roger Grosse
